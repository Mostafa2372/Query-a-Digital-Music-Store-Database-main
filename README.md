# Query-a-Digital-Music-Store-Database
In this project, we will query the Chinook Database. The Chinook Database holds information about a music store. For this project, we will be assisting the Chinook team with understanding the media in their store, their customers and employees, and their invoice information. the schema for the Chinook Database is provided below.
